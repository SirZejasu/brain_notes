# Definition
> [!question] ~
> Enthält die Methode <mark style="background: #BBFABBA6;">int compareTo(n)</mark>, womit Objekte verglichen werden.
> Aus der Collection Klasse kann <mark style="background: #BBFABBA6;">sort(List<T> list)</mark> zum sortieren der Liste verwendet werdem.
> 
> Collection Klasse:
> Für die richtige Sortierung ist es wichtig, die <mark style="background: #BBFABBA6;">compareTo</mark> Methode richtig zu definieren.
> ``` java
>public static <T extends Comparable<? super T>>
> void sort(List<T> list)
>```
