# Bindung
> [!question] ~
> Beziehung zwischen Aufrufausdruck , also der Syntax und der zur Laufzeit ausgeführten Methode, also der Semantik.

# Syntax 
```java
Objektausdruck.Methodenname(Expr 1, ..., Expr n)
Klasse.Methodenname(Expr 1, ..., Expr n)
```


# Statische Bindung
> [!question] ~
>  Zur Compilezeit wird festgelegt, welche Methode während der Laufzeit ausgeführt wird. Dabei wird nur eine mögliche Methode angewendet.

# Dynamische Bindung
> [!question] ~
> Bei einer Methode, die in mehreren Klassen verwendet wird, wird durch eine dynamische Bindung ermöglicht, dass die passende Methode relativ zum Typ des Objekts abgerufen wird (Nur zur Laufzeit!). Es wird dann die Methode ausgerufen, wo das Objekt entstammt.
> 
> In Zusammenhang mit der Klassenhierarchie ist Polymorphie möglich.
