# Definition
> [!question] ~
> Überprüft, ob der Wert eines Objekts in der Instanz einer Klasse oder dessen Unterklassen ist.
> ![[Pasted image 20230529124537.png]]

# Beispiel

>
>```java
>if (obj instanceof Buch b) {  
>
> istGleich = titel.equals(b.titel)  
>		&& autor.equals(b.autor)  
>		&& erscheinungsjahr == b.erscheinungsjahr;  
>
>}
>```
