# Aufgabe 3

```js
public static int quersumme(int zahl) { 

return (zahl < 10) 
? zahl 
: quersumme(zahl / 10) + (zahl % 10); 
}
```

```js
quersumme(4208)
= quersumme(zahl / 10) + (zahl % 10) zahl -> 4208
= quersumme(420) + (8)
= quersumme(428)
= quersumme(zahl / 10) + (zahl % 10) zahl -> 428
= quersumme(42) + (8)
= quersumme(50)
= quersumme(zahl / 10) + (zahl % 10) zahl -> 50
= quersumme(5) + 0
= quersumme(5)
= 5
```
