## Nicht-Einfacher Fall
Welches kleinere Problem der selben Art ist enthalten, sodass ich mit dessen Lösung auch das ursprüngliche Problem lösen kann?
```java
public static boolean istStellenweiseGroesser(long zahl1, long zahl2) {

	return (zahl1 < 10 && zahl2 < 10)
		? (zahl1 > zahl2)
		: istStellenweiseGroesser(zahl1 / 10, zahl2 / 10) && (zahl1 % 10 > zahl2 % 10);
		
}
```
