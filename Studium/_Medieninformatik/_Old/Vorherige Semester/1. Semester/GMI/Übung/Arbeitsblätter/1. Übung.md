# Aufgabe 5

## Aufgabe 5.1
$x^{5}+3x^{3}+2x$ Funktion hat keine Einschränkung.
$D=\mathbb{R}$

-f(x)=f(-x) (Ungerade)

$f(-x)=(-x)^{5}+3(-x)^{3}+2(-x)$
$\leftrightarrow-x^{5}-3x^{3}-2x$
$-f(x)=-(x^{5}+3x^{3}+2x)$
$\leftrightarrow-x^{5}-3x^{3}-2x$

Die Funktion ist ungerade.

## Aufgabe 5.2
$f(x)=\frac{\sin(x)}{x}$

$x=0$

$ID=\mathbb{R} \textbackslash \{0\}$

-f(x) = f(-x)

-f(x) = $-(\frac{sin(x)}{x})$
$\leftrightarrow -\frac{\sin(x)}{x}$

Es ist keine ungerade

$f(x)=f(-x)$

$f(-x)=\frac{\sin(-x)}{-x}$ (sin(-x)=-sin(x))
$\leftrightarrow \frac{-\sin(x)}{-x}\to$ Division von beiden negativen Zahlen ergibt immer eine positive Zahl.
$\leftrightarrow \frac{\sin(x)}{x}$ 

Es ist eine gerade.
