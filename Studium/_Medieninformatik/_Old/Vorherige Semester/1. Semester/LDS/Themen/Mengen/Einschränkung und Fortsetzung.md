# Definition
## Einschränkung
> [!question] Einschränkung
> Eine Einschränkung, schränkt eine Menge in bestimmten Elementen ein.
> Sei A und C Mengen, es wird geschrieben $A|C$, A wird auf Elemente von C beschränkt.

## Fortsetzung
> [!question] Fortsetzung
> Sei f und g eine Funktion. Man spricht von einer Fortsetzung, wenn $f \subseteq g$ ist dann ist g eine Fortsetzung von f.
