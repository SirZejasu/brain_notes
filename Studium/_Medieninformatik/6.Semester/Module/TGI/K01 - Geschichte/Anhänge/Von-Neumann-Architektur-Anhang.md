# Haupteinheiten der IAS
| Wort                                | Bedeutung                                                                                                |
| ----------------------------------- | -------------------------------------------------------------------------------------------------------- |
| Hauptspeicher (M)                   | Besteht aus 1000 Speicherplätzen, sogenannten **Wörtern**. Daten und Instruktionen werden hier abgelegt. |
| Arithmetisch logische Einheit (ALU) | Genau wie die Program Kontrolleinheit, enthält es verschiedene Register.                                 |
| Program Kontrolleinheit (CU)        | Ruft die Instruktionen aus dem Hauptspeicher und führt sie nach der Reihe aus.                           |
| I/O Geräte (IO)                     | Ein-/Ausgabe von Daten.                                                                                  |
