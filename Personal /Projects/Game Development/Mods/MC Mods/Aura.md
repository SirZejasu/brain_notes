# Beschreibung des Mods
Ein Minecraft Mod, dass eine mystische Ambiente im Spiel hinzufügt. Dabei geht es nicht um "machtvolle, zerstörerische Magie". Es soll möglichst zur Minecraft Ästhethik passen.
Es ist Wicca-inspiriert.

# Features
- New Plants.
- New Animals / Creatures.
- Celestial Events. (Mondphasen, Jahreszeiten, Wetter)
- When it rains, spirits come out.
- Summer = Highest Magic.
- Fall = Verringerte Magie.
- Winter = Kaum bis garkeine Magie.
- Frühling = Wiedergeborene Magie.
- Summer + Rain = A lot of fucking spirits.
